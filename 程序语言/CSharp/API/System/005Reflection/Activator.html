CreateInstance()方法通过调用与指定参数最匹配的构造函数来创建程序集中定义的类型的实例。如果没有指定参数，则调用无参数构造函数。

必须有足够的权限来搜索和调用构造函数，否则将抛出异常。默认情况下，在搜索构造函数时只考虑公共构造函数。如果找不到构造函数或无参数构造函数，则会抛出异常。

绑定参数指定了在程序集中如何搜索合适的构造函数的条件。如果没有指定绑定参数，则使用默认绑定参数。

可以在本地或远程站点创建类型的实例。如果类型是远程创建的，则激活属性参数指定远程站点的URI。创建实例的调用可能会在到达远程站点之前通过中间站点。其他激活属性可以修改调用在远程和中间站点运行的环境或上下文。

如果实例是在本地创建的，则返回对该对象的引用。如果实例是远程创建的，则返回对代理的引用。远程对象通过代理进行操作，就像它是本地对象一样。
