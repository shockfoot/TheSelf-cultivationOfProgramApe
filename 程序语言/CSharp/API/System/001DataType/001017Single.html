定义

C#

public readonly struct Single

命名空间：System

程序集：System.Runtime.dll

结构体只读

说明

Single（别名float）表示单精度浮点数，默认值为0，4字节，范围为-3.40282347E+38至3.40282347E+38。

Single的32位中，0至22位表示尾数，23至30位表示指数，31位为符号。因为有些数字不能精确地被二进制表示，所以浮点数只能近似表示实数。

C#

public int CompareTo(object? value)

比较当前实例与指定对象，并返回一个表明二者关系的整数：大于0表示当前实例大于指定对象，小于0则相反，等于0表示二者相等。需要注意的是，CompareTo(Object)的参数需要是Single类型或null，否则将会抛出异常。

因为不需要拆箱，CompareTo(Single)比CompareTo(Object)的性能更好。
