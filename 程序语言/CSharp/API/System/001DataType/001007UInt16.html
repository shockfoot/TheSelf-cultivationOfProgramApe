定义

C#

public readonly struct UInt16

命名空间：System

程序集：System.Runtime.dll

结构体只读

说明

UInt16（别名ushort）表示16位无符号整数，默认值为0，2字节，范围为0到65535（216-1）。

CompareTo

C#

public int CompareTo(ushort value)
public int CompareTo(object? value)

比较当前实例与指定对象，并返回一个表明二者关系的整数：大于0表示当前实例大于指定对象，小于0则相反，等于0表示二者相等。需要注意的是，CompareTo(Object)的参数需要是UInt16类型或null，否则将会抛出异常。

因为不需要拆箱，CompareTo(UInt16)比CompareTo(Object)的性能更好。
