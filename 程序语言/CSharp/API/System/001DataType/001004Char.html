定义

C#

public readonly struct Char

命名空间：System

程序集：System.Runtime.dll

结构体只读

说明

Char（别名char）表示Unicode UTF-16字符，默认值为\0，2字节，范围为0x0000到0xFFFF。

除了使用Char表示单个字符外，UTF-16编码还允许使用两个Char表示抽象字符，这被称为代理项对（Surrogate Pair）。其中前一个被称作高代理项（High Surrogate），代码点的范围为0xD800到0xDBFF；后一个被称作低代理项（Low Surrogate），代码点的范围为0xDC00到0xDFFF。单独的代理项没有意义，它只有在作为代理对的一部分使用时才有意义。

CompareTo

C#

public int CompareTo(char value)
public int CompareTo(object? value)

比较当前实例与指定对象，并返回一个表明二者编码值关系的整数：大于0表示当前实例在指定对象之后，小于0则相反，等于0表示二者编码值相等。需要注意的是，CompareTo(Object)的参数需要是Boolean类型或null，否则将会抛出异常。

因为不需要拆箱，CompareTo(Char)比CompareTo(Object)的性能更好。
