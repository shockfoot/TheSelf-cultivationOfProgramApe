定义

C#

public readonly struct Boolean

命名空间：System

程序集：System.Runtime.dll

结构体只读

说明

布尔类型Boolean（别名bool）表示一个布尔值，可为true或false，默认值为false，1字节。

字段

TrueString

C#

public static readonly string TrueString;

静态只读

此字段定义了Boolean值true的字符串表示形式，即"True"。

FalseString

C#

public static readonly string FalseString;

静态只读

此字段定义了Boolean值false的字符串表示形式，即"False"。

方法

Parse

C#

public static bool Parse(string value)
public static bool Parse(ReadOnlySpan value)

静态

将字符串转化为Boolean类型。需要注意的是，参数必须等效于TrueString或FalseString，不区分大小写，且前面和后面可以有空格，否则将会抛出异常。

TryParse

C#

public static bool TryParse(string? value, out bool result)
public static bool TryParse(ReadOnlySpan value, out bool result)

静态

与Parse方法类型，将字符串转化为Boolean类型，但不会抛出异常。

CompareTo

C#

public int CompareTo(bool value)
public int CompareTo(object? obj)

比较当前实例与指定对象，并返回一个表明二者关系的整数：大于0表示当前实例大于指定对象，小于0则相反，等于0表示二者相等。需要注意的是，CompareTo(Object)的参数需要是Boolean类型或null，否则将会抛出异常。

因为不需要拆箱，CompareTo(Boolean)比CompareTo(Object)的性能更好。
