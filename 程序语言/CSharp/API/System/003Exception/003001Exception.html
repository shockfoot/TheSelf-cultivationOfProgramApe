HResult

C#

public int HResult { get; set; }

获取和设置HRESULT。

HRESULT是一个分配给特定异常的编码，共32位，分为三个不同的字段：严重性编码、系统区域编码和错误编码。严重性编码表示信息、警告还是错误。系统区域编码标识了导致错误的系统区域。错误编码是一个唯一的数字，用来表示异常。每个异常都映射到一个不同的HRESULT。

当托管代码抛出异常时，运行时将HRESULT传递给COM客户端。当非托管代码返回错误时，HRESULT被转换为异常，然后由运行时抛出。

GetBaseException()

C#

public virtual Exception GetBaseException()

获取抛出当前异常的根本的异常实例。

异常链由一组异常组成，这样链中的每个异常都是作为其InnerException属性中引用的异常的直接结果抛出的。对于给定的链，可能只有一个异常是链中所有其他异常的根本原因。这个异常被称为基异常，它的InnerException属性总是包含一个空引用。对于异常链中的所有异常，此方法返回相同的实例。
