观察者模式也称作发布/订阅模式，观察者模式定义了一种一对多的依赖关系，让多个观察者对象同时监听某一个主题对象。这个主题对象在状态发生变化时，会通知所有观察者对象，使它们能够自动更新自己。此模式常用来实现事件处理系统。

特点：发布者+订阅者=观察者模式。

例子：野猫来了，老鼠A、B、C惊吓逃走。

``` C#
public class Animal // 动物基类
{
    protected string name;
    protected Animal (string name) { this.name = name; }
    public virtual void Run() { }
}
public class Mouse : Animal // 老鼠
{
    public Mouse (string name) : base(name) { }
    public override void Run()
    { Console.WriteLine("{0}开始跑", base.name); }
}
```

简单处理：

``` C#
public class Cat : Animal // 猫
{
    public Cat (string name) : base(name) { }
    public void Coming(params Mouse[] mouses)
    {
        Console.WriteLine("{0}来了", base.name); 
        foreach (Mouse mouse in mouses)
            mouse.Run();
        Run();
    }
    public override void Run()
    { Console.WriteLine("{0}开始追", base,name); }
}
// 测试
Cat cat = new Cat("小白");
Mouse mouseA = new Mouse("a");
Mouse mouseB = new Mouse("b");
Mouse mouseC = new Mouse("c");
cat.Coming(mouseA， mouseB, mouseC);
```

使用委托：猫是发布者，老鼠是监听者。

``` C#
public class Cat : Animal // 猫
{
    public Action mouseAction = null;
    public Cat (string name) : base(name) { }
    public void Coming()
    {
        Console.WriteLine("{0}来了", base.name); 
        if (mouseAction != null)
            mouseAction();
        Run();
    }
    public override void Run()
    { Console.WriteLine("{0}开始追", base,name); }
}
// 测试
Cat cat = new Cat("小白");
Mouse mouseA = new Mouse("a");
Mouse mouseC = new Mouse("c");
cat.mouseAction += mouseA.Run;
cat.mouseAction += mouseC.Run;
cat.Coming();
```
