# 固定函数着色器

固定函数着色器是针对硬件能够执行的命令编写的着色器，主要用于不支持可编程管线的设备，功能有限但速度快、支持性好。

``` ShaderLab
Shader "Learn/FixedFunctionShader"
{
    Properties {
        _Color ("Main Color", Color) = (1, 1, 1, 1)
    }
    
    SubShader
    {
        Pass
        {
            Material
            {
                Diffuse[_Color]
            }
        }
    }
}
```

固定函数着色器必须在Pass块中编写，通常是对通道的一些渲染设置，只能使用ShaderLab的渲染设置命令编写。
